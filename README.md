# TechPC Store

A custom PC e-commerce website built with PHP and MySQL.

## Tech Stack

- **PHP** - Server-side logic
- **MySQL** - Database
- **HTML/CSS** - Frontend
- **JavaScript** - Client-side interactions
- **Docker** - Containerization

## Deployment

- **Hosting:** [Render.com](https://render.com) - Web service hosting
- **Database:** [Aiven.io](https://aiven.io) - Managed MySQL database with SSL
